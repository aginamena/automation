# AI Automation Landing Page

A modern, responsive landing page for an AI automation business that helps small business owners reclaim their time by automating repetitive tasks.

## Features

- **Hero Section** - Compelling headline focused on business growth
- **About Section** - Personal bio and value proposition
- **Booking Integration** - Calendly embed for scheduling free strategy sessions
- **Session Details** - Clear outline of what happens during the consultation
- **Social Proof** - Testimonials from satisfied clients
- **Mobile Responsive** - Optimized for all device sizes
- **Minimal Design** - Clean, focused layout to maximize conversions

## Files

- `index.html` - Main landing page
- `styles.css` - All page styling and responsive design
- `README.md` - Project documentation

## Getting Started

Simply open `index.html` in a web browser to view the landing page.

To run locally with a server:

```bash
python -m http.server 8000
```

Then navigate to `http://localhost:8000`

## Customization

- Replace the Calendly embed URL in `index.html` with your own calendar link
- Update the email address in the footer
- Modify the video placeholder with your actual intro video
- Update testimonials with real client feedback
- Customize colors in `styles.css` via CSS variables in the `:root` section
